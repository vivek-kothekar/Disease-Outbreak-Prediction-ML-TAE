## Dataset: Diabetes Prediction Dataset

This folder contains the dataset **`diabetes.csv`**, used for the Disease Outbreak Prediction project, specifically focusing on diabetes risk prediction.

### 📌 Description

The dataset consists of medical and diagnostic information collected from patients to predict whether an individual is likely to have diabetes. It is widely used for classification tasks in machine learning and healthcare analytics.

### 📊 Features

The dataset includes the following attributes:

* **Pregnancies** – Number of times pregnant
* **Glucose** – Plasma glucose concentration
* **BloodPressure** – Diastolic blood pressure (mm Hg)
* **SkinThickness** – Triceps skin fold thickness (mm)
* **Insulin** – 2-Hour serum insulin (mu U/ml)
* **BMI** – Body Mass Index (weight in kg/(height in m)^2)
* **DiabetesPedigreeFunction** – Diabetes likelihood based on family history
* **Age** – Age of the patient
* **Outcome** – Target variable (0 = No Diabetes, 1 = Diabetes)

### 📁 File Information

* **File Name:** `diabetes.csv`
* **Format:** CSV (Comma Separated Values)
* **Type:** Structured tabular dataset
